# AI Resume Analyzer

## Overview

AI Resume Analyzer is a full-stack web application that analyzes resumes in PDF format and provides an ATS score, skill extraction, missing skills analysis, and job recommendations.

## Features

* Upload Resume PDF
* ATS Score Calculation
* Skill Extraction
* Missing Skills Detection
* Job Recommendations
* Responsive User Interface

## Tech Stack

### Frontend

* React.js
* JavaScript
* HTML
* CSS
* Axios

### Backend

* Python
* Flask
* Flask-CORS
* PyPDF2

## How It Works

1. Upload a PDF resume.
2. The backend extracts text from the PDF.
3. Skills are identified from the resume.
4. ATS score is calculated.
5. Missing skills are displayed.
6. Suitable job roles are recommended.

## Installation

### Backend

```bash
cd backend
pip install flask flask-cors PyPDF2
python app.py
```

### Frontend

```bash
cd frontend
npm install
npm start
```
